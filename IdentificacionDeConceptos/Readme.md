Descripción del programa
El programa sirve para identificar que tipo de triangulo se tiene a partir de las longitudes de este. 
El programa deberá revisar 3 valores de entrada y a partir de esto devolverá el tipo de triangulo que se introdujo.
Requerimientos
Triangulo.jar funciona bajo la plataforma java.
Su máquina debe tener instalado jre6 o superior para el funcionamiento correcto.
Abrir la consola y navegar donde se encuentra el archivo “Triangulo.jar” y ejecutar el comando “java -jar Trinagulo.jar”


Valores
Comunicación
El código del software debe ser fácil de entender para cualquier desarrollador. Para esto se debe escribir un código limpio y sencillo.

Simplicidad
Se debe evitar escribir código q resulte muy complejo ya que causa problemas ya que se difícil entender para otras personas involucradas, realizar cambios, etc. La complejidad depende de quien lo desarrolle ya que lo q es complejo para unos puede ser simple para otros. Por lo tanto, es importante pesar en los demás cuando se está escribiendo código. 

Flexibilidad
Cuando se desarrolla se lo debe hacer de tal manera que sea fácil realizar un cambio o añadirle una funcionalidad nueva.

Principios
Consecuencias locales
Se debe estructurar el código de manera que los cambios se realicen sobre este afecten localmente. Es decir que un cambio no debería afectar otras secciones del programa.

Minimizar repetición
Se debe evitar escribir el mismo código una y otra vez

Lógica y datos juntos
Se debe codificar de manera que los datos y lógica estén lo más juntos posibles para que las modificaciones que se hagan ya sea en la lógica o los datos afectan localmente como indica el principio de consecuencias locales.

Simetría
El código debe ser expresado de la misma manera en toda la aplicación.

Expresión declarativa
Consiste en expresar nuestro código sentencias que describa el problema que se quiere solucionar y no así los pasos para resolver dicho problema.

Ratio de cambio
Colocar los datos o lógica q cambian en el tiempo lo más juntos posible.
